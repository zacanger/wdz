# wdz

`words` and `unwords` functions.

[Donate](https://ko-fi.com/zacanger)

--------

## Installation

`npm i wdz`

## Usage

```javascript
const { words, unwords } = require('wdz')

words('foo bar baz') // => [ 'foo', 'bar', 'baz' ]
unwords(words('foo bar baz')) // => 'foo bar baz'
```
